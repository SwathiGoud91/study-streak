Study Streak – Gamified Learning Platform
🚀 Overview

Study Streak is a web-based gamified productivity platform designed to help students build consistent study habits. It tracks daily study activity, maintains streaks, provides motivational quotes, and includes interactive challenges like daily puzzles and leaderboards to boost engagement.

✨ Features
👤 User Registration & Login System
🔥 Daily Study Streak Tracking
📊 Personalized Dashboard
🧠 Daily Puzzle Challenges
🏆 Leaderboard System
💬 Motivational Quotes (daily refresh)
🥇 “Pro User” Badge System
📅 Activity Tracking System
🎯 Progress Monitoring
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript, Bootstrap
Backend: Python (Flask)
Database: SQLite
Hosting: Replit / Render (optional)
